Reference Guide

Columns

The dashboard shows five columns, each displaying stories from a different source. Columns are independently scrollable. Each competitor column (NYT, WSJ, WaPo) fetches via RSS feeds.

  • Perplexity Discover — Stories from Perplexity's Discover feeds (Top, Tech & Science, Finance). Filter by topic using the pills at the top. Cards are color-coded: teal for Top, purple for Tech & Science, gold for Finance.
  • NYT Homepage — Live RSS feed from The New York Times homepage.
  • WSJ (Consolidated) — Merges three WSJ feeds (World News, US Business, Markets) into one deduplicated column. Filter by section using the pills at the top.
  • Washington Post — Via Google News RSS proxy for washingtonpost.com.

Gap Indicators

When HIGHLIGHT GAPS is active, competitor stories that Perplexity isn't covering get a red dot badge and a red left border. These are stories present in competitor feeds but absent from Perplexity Discover.

Gap Analysis Logic

  • Client-side keyword extraction from titles
  • A competitor story is "missing" if <40% of its significant words match any Perplexity story
  • Op-eds are excluded (titles containing "Opinion", "Editorial", "Commentary", "Review", "Letters to the Editor")
  • Publisher normalization: All WSJ feeds count as 1 source, all NYT feeds as "NYT", etc.
  • Gaps are ranked by "buzz score" — stories mentioned by more publishers rank higher

Keyboard Shortcuts

  • R — Refresh All feeds
  • H — Toggle Highlight Gaps
  • G — Open/close Gap Analysis overlay
  • S — Open/close Social Pulse overlay
  • A — Open/close Add Sources overlay
  • I — Open/close Guide overlay
  • P — Quick Paste (manually add Perplexity stories)
  • Esc — Close any open overlay

Adding Custom Sources

Open Add Sources (A) to toggle existing sources on/off or add a custom RSS feed URL. Custom sources appear as additional columns.

Quick Paste — Add Perplexity Stories

Paste story titles (one per line) to add them to the Perplexity Discover column. Select a topic category below.
